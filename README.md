# pic2ascii
将图片转换成 ascii 图片的小工具
A tools for converting pictures into ASCII images

## Compile
pascal compiler

free pascal compiler
```
fpc pic2ascii.pas
```

## Usage
windows or linux console:
```
pic2ascii hello.jpg
pic2ascii hello.jpg hello.txt
```
